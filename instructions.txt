Rojira Projects Structure

[devel]/
	lib/
		<install dir for jars>
		<jogl jars>
		<jogl sos>
		<jsi4, sigl, siren jars>
	jmake5/
		build.sh
		compile.sh
		clean.sh
		run.sh
		install-lib.sh
	Jsi4/
		resource/
		src/
	SiGL/
		lib/
			xyz.jar -> ../../../lib/xyz.jar
		resource/
		src/
	Siren/
		lib/
			xyz.jar -> ../../../lib/xyz.jar
		resource/
		src/
	NYSF2014/
		lib/
			xyz.jar -> ../../../lib/xyz.jar
			xyz.jar -> ../../../lib/xyz.so 
		resource/
			JP5AppInfo
			NYSF2014.conf
			system1.conf
			...
		src/
		
JP5AppInfo is a shell script that is sourced but the compile.sh script
it has all the project specific variables for the jmake5 scripts to use	

you can specify any number of systems to run symultaneously in the resources folder

cd [devel]/Jsi4
	../jmake5/compile.sh
	../jmake5/install-lib.sh
	
cd [devel]/SiGL
	../jmake5/compile.sh
	../jmake5/install-lib.sh
	
cd [devel]/Siren
	../jmake5/compile.sh
	../jmake5/install-lib.sh
	
cd [devel]/NYSF2014
	../jmake5/compile.sh
	../jmake5/run.sh
